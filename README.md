# Hue Slide Demo

スマートフォン向けのシンプルな色遊びデモ（依存ライブラリなし・PWA 対応）。

- ランダムな色で全画面を塗りつぶし（**HSV: 彩度 80% / 明度 80%** から開始）
- **左右スワイプで色相を黄金角（≈137.5°）ずつ変更** — 毎回はっきり違う色が出て、色相環に偏りなく分散。左右スワイプは互いに厳密な逆操作
- 遷移は**なめらかな波打つ境目のワイプ**（`clip-path`）。波形は**毎回ランダム**に生成され、着地時に弾性リコイル。中間色を経由しないのでチラつかない
- **上下スワイプで彩度(S)を変更**（上=鮮やか／下=くすむ）。切替は横とは別系統の**粒状ディゾルブ**演出
- 切替時に **HSB由来のサウンド**（Web Audio）: Hue→ペンタトニック音高＋グリッサンド、S→音色の明るさ、V→音量
- 画面右上に現在の色相を示す**円形インジケータ＋カーソル＋数値**（**ダブルタップで表示/非表示**）
- **PWA**（`display: fullscreen`）としてインストールするとフルスクリーン起動

## ローカルで動かす

PWA / Service Worker は `file://` では動かないため、HTTP で配信してください。

```sh
python -m http.server 8123
# → http://localhost:8123/
```

インストール（ホーム画面に追加）や実機での Service Worker には HTTPS または `localhost` が必要です。

## 操作

- 左スワイプ: 色相を +137.5°（次へ）／右スワイプ: −137.5°（前へ）
- 上スワイプ: 彩度アップ（鮮やか）／下スワイプ: 彩度ダウン（くすむ）
- ダブルタップ: 右上のインジケータを表示/非表示
- PC: キーボードの ← / → / ↑ / ↓

## ファイル構成

- `index.html` — 本体（CSS/JS インライン）
- `manifest.webmanifest` — PWA マニフェスト
- `sw.js` — Service Worker（ネットワーク優先＋オフラインフォールバック）
- `icons/` — アプリアイコン（192 / 512 / maskable）
